## Installation

```bash
#Download the Flumbot
git clone https://www.github.com/Baconmaster326/Flumbot
#Get the Flumbot Dependencies
uv sync
#Run the Flumbot
python FlumbotGOTEdition.py
```

### Prerequisites

The following will need to be installed and added to environment variables. 
- fluidsynth
- yt-dlp

a token.json file containing api keys is also necessary. As well as a cookies.txt file for downloading content from 
YouTube.
